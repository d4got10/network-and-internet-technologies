1. Accept-Language - Список поддерживаемых естественных языков.
2. Accept - Список допустимых форматов ресурса.
3. Authorization - Данные для авторизации.
4. Proxy-Authorization - Информация для авторизации на прокси-сервере.
5. From - Адрес электронной почты ответственного лица со стороны клиента.
6. Host - Доменное имя и порт хоста запрашиваемого ресурса. Необходимо для поддержки виртуального хостинга на серверах.
7. Expect - Указывает серверу что клиент ожидает от него дополнительного действия.
8. Max-Forwards - Максимально допустимое количество переходов через прокси.
9. User-Agent - Список названий и версий клиента и его компонентов с комментариями.
10.Referer - URI ресурса, после которого клиент сделал текущий запрос.
