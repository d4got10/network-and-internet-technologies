1. Accept-Ranges - Перечень единиц измерения диапазонов.
2. WWW-Authenticate - Параметры аутентификации для выполнения метода к указанному ресурсу.
3. Retry-After - Дата или время в секундах после которого можно повторить запрос.
4. Server - Список названий и версий веб-сервера и его компонентов с комментариями. Для прокси-серверов поле Via.
5. Alternates - Указание на альтернативные способы представления ресурса.
6. Age - Количество секунд с момента модификации ресурса.
7. Vary - Список описывающих ресурс полей из запроса, которые были приняты во внимание.
8. Proxy-Authenticate - Параметры аутентификации на прокси-сервере.
9. Public - Список доступных методов аналогично Allow, но для всего сервера.
10.Location - URI по которому клиенту следует перейти или URI созданного ресурса.
